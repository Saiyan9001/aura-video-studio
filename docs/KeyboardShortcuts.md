# Keyboard Shortcuts

Master these keyboard shortcuts to work faster and more efficiently in Aura Video Studio.

## Timeline Controls

These shortcuts work when you're in the **Storyboard** view with the timeline focused.

### Playback Controls

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Space** | Play/Pause | Toggle playback of the timeline |
| **J** | Rewind | Play backward (shuttle reverse) |
| **K** | Pause | Stop playback instantly |
| **L** | Fast Forward | Play forward (shuttle forward) |
| **S** | Stop | Stop playback and return to start |

**Tip**: Use J/K/L for precise control when reviewing your video. Tap multiple times to increase shuttle speed.

### Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Left Arrow** | Previous Frame | Step backward one frame |
| **Right Arrow** | Next Frame | Step forward one frame |
| **Home** | Go to Start | Jump to beginning of timeline |
| **End** | Go to End | Jump to end of timeline |
| **Q** | Previous Scene | Jump to previous scene marker |
| **W** | Next Scene | Jump to next scene marker |

### Zoom and View

| Shortcut | Action | Description |
|----------|--------|-------------|
| **+** (Plus) | Zoom In | Zoom into timeline for detail |
| **-** (Minus) | Zoom Out | Zoom out for overview |
| **Ctrl + 0** | Zoom to Fit | Fit entire timeline in view |
| **Ctrl + 1** | Zoom 100% | Reset to default zoom level |

**Tip**: Zoom in to precisely adjust scene timing and markers.

### Editing

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl + Z** | Undo | Undo last action |
| **Ctrl + Y** | Redo | Redo previously undone action |
| **Ctrl + X** | Cut | Cut selected element |
| **Ctrl + C** | Copy | Copy selected element |
| **Ctrl + V** | Paste | Paste from clipboard |
| **Delete** | Delete | Delete selected scene or element |

### File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| **Ctrl + S** | Save | Save current project |
| **Ctrl + Shift + S** | Save As | Save project with new name |
| **Ctrl + O** | Open | Open existing project |
| **Ctrl + N** | New | Create new project |

## Tips for Efficiency

### Learn the Shuttle Controls (J/K/L)
The J/K/L keys are industry-standard for video editing:
- Tap **J** multiple times to rewind faster (up to 3x speed)
- **K** always stops playback instantly
- Tap **L** multiple times to fast forward faster (up to 3x speed)
- Hold **K** + tap **J** or **L** for frame-by-frame review

### Quick Scene Navigation (Q/W)
Jumping between scenes with **Q** (previous) and **W** (next) is much faster than scrolling:
- Review each scene's content quickly
- Check transitions between scenes
- Verify timing consistency

### Zoom for Precision (+/-)
- Zoom in (**+**) when adjusting fine timing
- Zoom out (**-**) to see overall pacing
- Use **Ctrl + 0** to fit entire timeline

### Save Often (Ctrl + S)
Get in the habit of saving frequently:
- After major changes
- Before rendering
- Before closing the application
- Auto-save runs every 5 minutes (configurable)

## Customizing Shortcuts

Want to change keyboard shortcuts?

1. Go to **Settings** → **Keyboard**
2. Click on any shortcut to edit
3. Press your desired key combination
4. Click **Save** to apply changes
5. Click **Reset to Defaults** to undo all changes

**Note**: Some shortcuts cannot be changed as they're system-level (like Ctrl+C/V/X).

## Shortcut Conflicts

If a shortcut doesn't work:
1. Check if another application has captured it (e.g., gaming overlays)
2. Verify you're in the correct view (some shortcuts are context-specific)
3. Try clicking in the timeline or view first to focus it
4. Check **Settings** → **Keyboard** for conflicts
5. Restart the application if needed
